Esta carpeta contiene todo el código en Pascal para la implementación mínima de las colas múltiples.

Únicamente tenemos las funciones:
  - initialize
  - isArrQueueEmpty
  - next
  - enqueueArr
  - dequeueArr

El programa consta de 4 archivos: uPersona.pas, uColaPersona.pas, uControlCola.pas, y main.pas.
